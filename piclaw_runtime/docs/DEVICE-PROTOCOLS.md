# Device Protocol Matrix

This file is maintained by the LAN/device discovery pipeline.

Current adapters:

- `onvif`: camera metadata/profile scaffold
- `rtsp`: stream URL resolver
- `chromecast`: playback scaffold
- `airplay`: playback scaffold
- `dlna`: playback scaffold
- `ssh`: remote command runner

Run `/devices` and `/device_show` on Telegram or `piclaw devices list` via CLI to inspect discovered capabilities per device.
